
# Instructions

##  To build journal:

- `make`      for defaults
- `make all`  the defaults (pdf and html)
- `make pdf`  ... 
- `make html` ...
- `make odt`  ...


## Format

Filenames should be numerical sorted:

- `./YYYY/YYYYMMDD_0000.md`
- `./YYYY/YYYYMMDD.md`


File template:

-----

```
## 2022-02-21 00:00 [USERNAME]

[//]: # (Keywords: #key_1, #key_2)

Text for this entry

```
